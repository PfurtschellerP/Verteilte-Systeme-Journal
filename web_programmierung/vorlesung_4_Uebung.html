Nun kommen wir zu unserer ersten benoteten Übung. Es soll ein Währungsumrechner programmiert werden, aber mehr dazu aus der original Aufgabenstellung

meine Umsetzung

Währungsumrechner

Code und Limitierungen

Das oben gezeigte Programm hat leider ein paar Schwächen. So sind die Umrechnungsfaktoren hardcoded und die Eingabe von Zeichenfolgen führt zu NaN(Not a number)-Fehlern. Aber nun zum Code. Der HTML Code für das Formular ist der folgende: 

                            
    <form id="waehrungsrechner" name="waehrungsrechner">
        <h2>Währungsumrechner</h2>
        <label id="eingabe">
            Eingabe
            <input type="text" name="eingabe" value="1">
        </label>
        <label id="umrechenAuswahl">
            <label>
                <input type="radio" name="umrechenAuswahl" checked>
                Euro nach USD
            </label>
            <br>
            <label>
                <input type="radio" name="umrechenAuswahl">
                USD nach Euro
                </label>
        </label>
        <input id="umrechnenButton" type="button" value="Umrechnen" onclick="umrechnen()">
        <h2 id="ausgabe"></h2>
    <form>

Es handelt sich hier um ein einfaches Formular mit einem text input sowie zwei radio buttons und einem button, welcher die Funktion umrechnen() aufruft. Den Code der Funktion umrechnen() kann man hier unten sehen: 

                            
    function umrechnen(){
        var umrechenfaktor = 0.0;
        var von = "";
        var zu = "";
        var eingangsWert = parseFloat(document.waehrungsrechner.eingabe.value.replace(",","."));
        var ausgangsWert = 0.0;
    
        // Wenn Euro zu USD ausgewählt ist
        if (document.waehrungsrechner.umrechenAuswahl[0].checked) {
            umrechenfaktor = 1.1698;
            von = "Euro";
            zu = "USD";
        // Wenn USD zu Euro ausgewählt ist
        } else if (document.waehrungsrechner.umrechenAuswahl[1].checked){
            umrechenfaktor = 0.8548;
            von = "USD";
            zu = "Euro";
        }
        ausgangsWert = eingangsWert * umrechenfaktor;
    
        document.getElementById('ausgabe').innerHTML = eingangsWert.toFixed(2).replace(".",",") + " " + von + " entsprechen " + ausgangsWert.toFixed(2).replace(".",",") + " " + zu;
    }

Wir ziehen und hier bei der Ausführung den eingegebenen Wert aus dem Textfeld und führen eine Fallunterscheidung durch. Je nachdem welcher radio button ausgewählt ist wird der Umrechenfaktor anders gewählt und der Ausgangswert umgerechnet.
Anschließend wird das Ergebnis innerhalb eines h2-Tags ausgegeben.
