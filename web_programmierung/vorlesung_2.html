Einbinden von JS in HTML

Um JavaScript Code in HTML einzubinden gibt es analog zu CSS mehrere Möglichkeiten. Die zwei gebräuchlichsten sind das Einfügen eines script-Tags mit JavaScript Code im HTML Dokument selbst und das EInfügen eines script-Tags aber mit gleichzeitigem Auslagern des Quellcodes wie hier demonstriert: 

                            
    <script src="script.js"></script>

Variablen

Nun wurde bereits in dr ersten Vorlesung angesprochen, dass JavaScript mit dynamischer Typenprüfung arbeitet. Was das genau bedeutet wurde und an einem Beispiel erklärt. 

                            
    var x = "5";
    var y = 15;
    alert(x+y);

Denn das Ergebnis hier ist nicht (wie man vermuten würde) 20, sondern 515. Das hängt damit zusammen, dass JavaScript entscheidet, um welchen Variablentyp es sich hier handelt. In diesem Fall wird String vermutet. Somit wird die Variable y auch in String konvertiert und die beiden Variablen (nun Zeichenfolgen) werden verkettet.

DOM Modell

Wie der letzte Abschnitt bereits demonstriert hat lässt sich mit JavaScript der Inhalt einer HTML Site verändern. Möglich macht dies das Document Object Model, kurz DOM. Man kann sich dieses wie ein umgedrehtes Baum Diagramm vorstellen in welchem dich die einzelnen Elemente der Website nach unten hin verzweigen. Mit Javascript können wir diese "Zweige" ansprechen (sogar noch besser, wenn wir ihnen ID's geben). So konnten wir im Beispiel oben weitere Zeilen generieren und zum HTML Element hinzufügen.
Wie dies konkret funktioniert ist im Code dargestellt.

Zahlen raten

Abschließend haben wir noch ein kleines Spiel in JavaScript programmiert. Ziel des Spieles ist es, die zufällig generierte Zahl zu erraten. Dazu bekommt der Spieler nach Abgabe eines Tipps Hinweise (ob er zu hoch oder zu niedrig getippt hat). 

                            
    function zahlenRaten(){
        var zahl = Math.round(Math.random()*100 +0.5);
        var fehler = 0;
        var fertig = false;

        while (!fertig){
            eingabe = prompt("Bitte erraten Sie die Zahl:","");
            if (!eingabe){
                fertig = true; //Abbruch durch Anwender
            } else{
                if (eingabe == zahl){
                    alert('Gewonnen!');
                    alert('Es wurden ' + fehler + " gemacht")
                    fertig = true; //Zahl wurde erraten
                } else{
                    fehler++;
                    if (eingabe < zahl){
                        alert('Geratene Zahl zu klein!\nGemachte Fehler: ' + fehler);
                    } else {
                        alert('Geratene Zahl zu groß!\nGemachte Fehler: ' + fehler);
                    }
                }
            }
        }
        var ausgabe = document.getElementById('zahlenRatenAusgabe');
        ausgabe.innerHTML = "Spielende!";
    }

Hausaufgaben

Am Ende dieser Vorlesung haben wir auch zwei kleine Hausaufgaben bekommen. Diese finden sich hier runter:

Dreieck

Schreiben Sie ein Programm, das mit Hilfe geschachtelter Schleifen ein aus *-Zeichen zusammengesetztes Dreieck auf der Konsole ausgibt. Der Benutzer soll vorher nach der Anzahl der Zeilen gefragt werden.

Der Quellcode dafür sieht dann folgendermaßen aus: 

                            
    function dreieck(){
        var dreieckAusgabe = document.getElementById('dreieckAusgabe');
        var anzahl = prompt("Bitte Anzahl angeben:","");
    
        for(var i = 1; i <= anzahl; i++){
            for(var j = 1; j <= i; j++){
                console.log("*");
                dreieckAusgabe.innerHTML += "*";
            }
            dreieckAusgabe.innerHTML += "<br>";
        }
    }

Das kleine 1 mal 1

In der nächsten Aufgabe soll das kleine 1 mal 1 als Tabelle ausgegeben werden.

Der Code hierfür ist etwas komplexer also breche ich ihn kurz in seine Einzelteile herunter. Wir starten damit, dass wir unser div Element im HTML auswählen. Danach bearbeiten wir das DOM Model indem wie ein Child item, in diesem Fall ein table-Element anhängen.
Nun kommen die for-Schleifen zum Einsatz. Wir durchlaufen hier de erste Schleife zehn mal. Dadurch das wir nun aber auch die zweite Schleife jeweils zehn mal durchlaufen ergeben sich insgesamt hundert Schleifendurchläufe. Das einmal eins ergibt sich dabei aus der Multiplikation der jeweiligen Laufvariablen. Der Kniff war hier letztendlich die Struktur der Tabelle weiter zu pflegen, also in diesem Fall die Referenz auf das aktuelle Reihen-Element reihe in die innere Schleife weiterzugeben um dort weitere Spalten erstellen zu können. 

                            
    function einMalEins(){
        var einMalEinsAusgabe = document.getElementById('einMalEinsAusgabe');
        // Tabelle erschaffen
        var tabelle = einMalEinsAusgabe.appendChild(document.createElement("table"));
    
        for(var i = 1; i <= 10; i++){
            // Reihe hinzufügen
            var reihe = tabelle.appendChild(document.createElement("tr"));
            for(var j = 1; j <= 10; j++){
                // Spalte hinzufügen
                spalte = reihe.appendChild(document.createElement("td"))
                spalte.innerHTML = i*j;
            }
        }
    }
