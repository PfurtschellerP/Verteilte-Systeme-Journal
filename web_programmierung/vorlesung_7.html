Kleine Übung in der Vorlesung

Nun haben wir eine kleine Übung gemacht. Wir sollten einen Streifen Effekt über Überschriften legen.
Ich hab das hier mal auf h2 Überschriften angewendet:

Eine H2-Überschrift

Noch eine

Und noch eine

Erreicht wird dies hier unter Verwendung von JQuery. Dabei wird zur Laufzeit ein span-Tag for jedes h2-Tag gesetzt, welches einen Strefen Hintergrund besitzt. Den Code dazu sieht man hier: 

                            
    .streifenEffekt{
        position: absolute;
        width: 15rem;
        display: block;
        height: 36px;
        background: url("gradient.png") ;
    }
                            
                        
                            
    $(document).ready( function(){
        $("#uebung h2").prepend('<span class="streifenEffekt">')
    })

Der Code dazu sieht folgendermaßen aus: 

                            
    .pullquote{
        float: right;
        clear: right;
        display: block;
        width: 20vw;
        border: 1px solid black;
        padding: 1vh;
        font-size: 1.2rem;
        background-color: #FFC;
        margin: 0.2rem 0 0.2rem;
        font-style: italic;
        color: #363;
    }
                            
                        
                            
    $(document).ready( function(){
        // für jedes span-Tag mit der Klasse pq
        $("span.pq").each(function(){
            // Element duplizieren
            var quote = $(this).clone();
            // CSS Klasse austauschen
            quote.removeClass("pq");
            quote.addClass("pullquote");
            // dubliziertes Element in den HTML Code "pflanzen"
            $(this).before(quote);
        })
    })
